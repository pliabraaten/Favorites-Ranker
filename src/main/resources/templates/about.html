About Favorites Ranker

A passion project built from scratch

The Ranking Algorithm

Favorites Ranker uses Binary Insertion Sort with pairwise comparisons to build your ranking interactively. Binary search minimizes the number of questions you need to answer.

Complexity Analysis
  • Comparisons: O(n log n) (Binary search per item)
  • Insertions: O(nÂ²) (Array shift operations dominate runtime)

Result: Ranking 20 items with direct comparisons would require 190 questions. Binary insertion sort reduces this to just ~86 comparisons, that's 55% fewer questions!
